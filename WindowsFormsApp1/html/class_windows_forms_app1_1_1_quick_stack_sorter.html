Подробное описание

Реализация сортировки для элементов типа Т. Использует алгоритм быстрой сортировки. Основан на стековой реализации алгоритма. Наследуется от ISorter.

Функция, определяющая индекс элемента, разделяющего массив на две части. Меняет местами два элемента, таким образом, что они казываются в порядке возрастания.
