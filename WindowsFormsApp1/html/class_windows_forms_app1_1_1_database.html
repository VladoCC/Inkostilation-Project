Класс WindowsFormsApp1.Database

Класс-оболочка базы данных, предоставляющий доступ ко всем меодам, необходимым для работы со структурами данных. Реализует шаблон проектирования Singleton. Предоставляет возможности для сохранения и загрузки базы данных на жесткий диск, при помощи сериализации.
